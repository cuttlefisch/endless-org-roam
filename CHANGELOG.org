* Unreleased

** Added
- Package skeleton with =eor-mode= global minor mode
- Instance registry (=eor-register-instance=, =eor-registry-get=, =eor-registry-remove=)
- Sentinel node (=eor-instance.org=) for idempotent registration
- =eor:= link type with local-first resolution algorithm
- Local transport backend (cross-instance DB queries via let-binding)
- Circuit breaker for remote instance availability tracking
- Compatibility module (=endless-org-roam-compat.el=)
- Cross-instance search skeleton (=eor-node-find=, =eor-node-insert=)
- Buttercup test suite (registry, link, transport)
- CI workflow (Emacs 28.1, 29.4, 30.1, snapshot)
- Contributing guidelines, Code of Conduct, PR/issue templates
